Kiti atvejai - if ... elif naudojimas

If sakiniai dažnai įtraukia bent du atvejus, o kartais ir daug daugiau. Tokiu atveju naudojama elif arba else if struktūra. Lietuviškai šią sakinio dalį išverstume "... kitu atveju, jeigu ...". Naudojimas jums paaiškės pamačius pavyzdžius, o kol kas panagrinėkime šios struktūros savybes:

  • elif / else if leidžia tikrinti kelias sąlygas (vieną po kitos).
  • Galima naudoti tiek elif blokų kiek tik reikia.
  • Kiekviena elif dalis atsakinga už naujos sąlygos apdorojimą.
  • Jei įvertinant skirtingas sąlygas yra randama teisinga, tuomet bloke esantis kodas yra įvykdomas ir tolesnės sąlygos nebetikrinamos.

Sintaksė

if salyga_1:
    # šio bloko kodas vykdomas jeigu salyga_1 reikšmė yra True
elif salyga_2:
    # šio bloko kodas vykdomas jeigu salyga_2 reikšmė yra True
elif salyga_3:
    # šio bloko kodas vykdomas jeigu salyga_3 reikšmė yra True
