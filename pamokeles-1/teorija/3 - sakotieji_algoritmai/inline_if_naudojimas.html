Patikrinimas vienoje eilutėje (inline if / ternary operator)

If sakinį galime sutrumpinti panaudodami ternarinį operatorių. Taip kelių eilučių if sakinį galime užrašyti viena eilute.

Sintaksė:

[ką daryti True atveju] if [patikrinimo sąlyga] else [ką daryti False atveju]

Tarkime, turime tokią užduotį: jeigu pažymys yra 10, tai turi būti atspausdinama "Puikai", o jeigu pažymys yra kitoks, tai turi būti atspausdinama "Galima geriau".

Kaip matome - sutaupėme pora eilučių. Vis tik - reikėtų atsargiai naudoti ternarinius operatorius. Programuotojai lig šiol diskutuoja ar verta juos naudoti. Manoma, jog jie gali sumažinti programos skaitomumą ir įtakoti klaidų atsiradimą.
