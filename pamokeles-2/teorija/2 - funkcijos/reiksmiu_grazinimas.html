Funkcijos reikšmų gražinimas

Kaip ir daugelyje programavimo kalbų, Pitone naudojamas raktažodis return funkcijoje apskaičiuotų reikšmių gražinimui. Prisiminkime funkcijos sintaksę:

def funkcijos_pavadinimas():
    # funkcijos kodas
    return grazinamas_atsakymas

Paprasčiausias būdas pagriebti funkcijos apskaičiuotą reikšmę (ar kelias reikšmes!) - tai funkcijos rezultatą priskirti kintamajam:

atsakymas = funkcijos_pavadinimas()
print(atsakymas)

Bet... tai nėra visuomet būtina. Kartais galime vieną funkciją naudoti kaip kitos funkcijos parametrą. Čia funkcija print pirmiau iškvies funkcijos_pavadinimas ir gautą rezultatą iš karto atspausdins.

Pavyzdžiai

Pradėsime nuo paprasto pavyzdžio, kur funkcija suformuoja pasisveikinimo tekstą:

Kaip matote - viskas paprasta: suformuojame reikšmę ir ją gražiname. Visgi - kartais jums prisireiks apskaičiuoti ne vieną, o kelias reikšmes. Kaip tada? Pitone tai išspręsta paprastai - gražiname reikšmes atskirdami jas kableliais.

Parašykime supaprastintą (neįvertinant galimų kompleksinių reikšmių) kvadratinės lygties skaičiavimo funkciją, kuri gražina dvi reikšmes.
